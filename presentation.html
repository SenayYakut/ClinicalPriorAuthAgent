04 — Architecture
Built on Azure AI
AI
Azure OpenAI GPT-4.1
Multi-step clinical reasoning with function calling. 5 tools orchestrated in sequence for end-to-end case processing.
R
RAG Pipeline
TF-IDF vector search over 8 payer policy documents across 3 insurance companies. Cosine similarity retrieval.
H
Human-in-the-Loop
Automatic routing to clinical reviewers when confidence drops below 70%. No autonomous decisions on borderline cases.
A
Audit & Compliance
Full agent trace logged for every case. Every tool call, input, and output recorded for HIPAA compliance and transparency.
05 — Responsible AI
Trust by design, not afterthought
  • Human-in-the-loop: Cases below 70% confidence are routed to clinicians — the agent never auto-approves uncertain cases
  • Transparency: Every agent decision is traceable — full tool execution history visible to reviewers
  • Gap analysis: Agent proactively identifies missing documentation and flags risks before submission
  • No hallucinated policies: RAG retrieval grounds the agent in actual payer policy documents, not generated content
  • Confidence scoring: Quantified prediction with strengths, risks, and recommendations — not a black box
